Detailed Description

Packet containing compressed media data (audio or video).

Member Function Documentation

◆ codec()

virtual const char* nx::sdk::analytics::ICompressedMediaPacket::codec ( ) const
pure virtual
Returns
Null-terminated ASCII string containing the MIME type corresponding to the packet data compression. For the video data, the following MIME types are available:
  • "video/h264" for H264 streams
  • "video/hevc" for H265/HEVC streams
  • "video/mpeg4" for MPEG-4 streams
  • "video/mjpeg" for MJPEG streams

For the metadata, check the Onvif documentation.
